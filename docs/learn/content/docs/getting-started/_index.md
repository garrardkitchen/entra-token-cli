# Getting Started

Welcome! This section will help you install Entra Auth Cli and generate your first token.

## What You'll Learn

- How to install the tool on your platform
- Creating your first authentication profile
- Generating and using access tokens
- Understanding the basics of OAuth2 flows

## Prerequisites

Before you begin, ensure you have:

- **.NET Runtime** (for global tool) or download self-contained executable
- **Azure/Entra ID Account** with access to app registrations
- **App Registration** (or permission to create one)

Don't have an app registration yet? No problem! The [First Token Tutorial](/docs/getting-started/first-token/) walks you through the complete setup.
